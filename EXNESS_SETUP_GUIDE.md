## 📱 المرحلة الأولى: إنشاء وإعداد بوت تليجرام (Telegram Bot Setup)

للحصول على إشعارات الصفقات، التقارير الدورية، وتنبيهات المخاطر على هاتفك:

### 1. إنشاء البوت والحصول على الـ Token:
1. افتح تطبيق تليجرام وابحث عن المستخدم **`@BotFather`**.
2. أرسل الأمر `/newbot`.
3. سيسألك عن اسم البوت (مثل: `Alpha Quant Trading Bot`).
4. سيسألك عن يوزر نايم للبوت ينتهي بـ `bot` (مثل: `my_alpha_quant_bot`).
5. سيعطيك BotFather رسالة تحتوي على **API Token** بالشكل التالي:
   `123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ` (احتفظ بهذا الرقم).

### 2. الحصول على Chat ID الخاص بك:
1. ابحث في تليجرام عن البوت **`@userinfobot`**.
2. أرسل له أي رسالة أو اضغط `/start`.
3. سيرد عليك برقم يمثل **`Id`** الخاص بك (مثل: `987654321`).

### 3. إدخال البيانات في إعدادات المشروع:
قم بفتح ملف الإعدادات [settings.py](file:///c:/Users/BASEEL%20PC/OneDrive/Desktop/%D8%AA%D8%B5%D9%85%D9%8A%D9%85%20%D8%A7%D9%84%D9%81%D8%A7/alpha_platform/config/settings.py) أو إنشاء ملف `.env` وضبط المتغيرات:
```env
TELEGRAM_BOT_TOKEN="123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ"
TELEGRAM_CHAT_ID="987654321"
```

## 📈 المرحلة الثانية: إعداد منصة Exness MetaTrader 5 (MT5)

### 1. تثبيت المنصة وتجهيز الحساب:
1. قم بتحميل منصة **Exness MetaTrader 5** وتثبيتها.
2. سجّل الدخول بحساب التداول الخاص بك:
   * **Account Number (Login)**
   * **Password**
   * **Server Name** (مثل `Exness-Real9` أو `Exness-Trial5`).
3. ينصح ببدء التجربة على **حساب تجريبي (Demo Account)** أولاً.

### 2. تفعيل التداول الآلي (Algo Trading):
1. افتح منصة MT5.
2. اذهب إلى القائمة العلويّة: `Tools` ⬅️ `Options` (أو اضغط `Ctrl + O`).
3. اختر تبويب **`Expert Advisors`**.
4. ضع علامة صح ✅ على **`Allow Algo Trading`**.
5. ضع علامة صح ✅ على **`Allow DLL imports`**.
6. اضغط **OK**.

### 3. تثبيت مكتبة Python MT5:
في السيرفر أو الكومبيوتر، يتم تثبيت مكتبة الربط المباشر عبر الأمر:
```bash
pip install MetaTrader5
```

## 🖥️ المرحلة الثالثة: إعداد سيرفر VPS للعمل 24/7 بدون كومبيوتر

لكي يعمل البوت بدون الحاجة لإبقاء جهازك الشخصي يعمل:

### 1. اختيار مواصفات السيرفر (Windows VPS):
* **نظام التشغيل:** Windows Server 2019 أو Windows Server 2022.
* **المواصفات الموصى بها:** 2 vCPU / 4GB RAM / 50GB SSD.
* **أبرز الشركات المزودة:**
  * **Contabo VPS** (اقتصادي وجيد جداً ~ 10-12$ شهرياً).
  * **ForexVPS** (مخصص للتداول وسريع جداً ~ 15-20$ شهرياً).
  * **Vultr / Kamatera** (دفع بالدقيقة وسريع).

### 2. الاتصال بالسيرفر:
1. من جهازك الكومبيوتر (Windows)، ابحث عن برنامج **`Remote Desktop Connection`** (اتصال سطح المكتب البعيد).
2. أدخل عنوان الـ **IP** الخاص بالسيرفر ثم اسم المستخدم `Administrator` وكلمة السر الممنوحة لك من شركة الـ VPS.
3. ستفتح لك شاشة كومبيوتر حقيقي يعمل في السحابة.

### 3. نقل الملفات وتشغيل المنصة على الـ VPS:
1. قم بتثبيت Exness MT5 على الـ VPS وسجّل الدخول بحسابك.
2. قم بنسخ مجلد مشروعك `تصميم الفا` إلى السيرفر.
3. قم بتشغيل البرنامج عبر تشغيل ملف `run_platform.bat`.
